## Overview

Nexus Ones is a Personal Operating Hub designed to unify daily planning, task management, reminders, notes, projects, courses and plugins into a single workspace.

### Core Modules

| Module | Description |
|----------|----------|
| Calendar | Calendar, Lunar Calendar, Holidays |
| Tasks | Task Planning and Tracking |
| Reminders | One-time and Recurring Reminders |
| Notes | Quick Notes and Personal Records |
| Projects | Project Management |
| Plugin Center | Plugin Discovery and Management |
| Update Center | Version Management |
| Data Center | Local Data Management |
