## 产品定位

Nexus Ones 不是单一的日历、待办或备忘录工具。

它是一套面向个人长期使用的办公中枢系统，用于整合日常工作、课程安排、项目推进、提醒事项、个人记录与插件扩展，让分散的信息回到一个统一入口。

---

## 核心能力

| 模块 | 说明 |
|---|---|
| Calendar | 日历、月视图、年视图、农历、节假日与事项管理 |
| Tasks | 待办事项、完成状态与任务追踪 |
| Reminders | 一次性提醒与循环提醒 |
| Notes | 快速备忘与个人记录 |
| Projects | 项目管理与事项归档 |
| Course Center | 课程表与教学安排管理 |
| Plugin Center | 插件发现、启用、禁用与插件生态扩展 |
| Update Center | 版本检查、更新说明与发布管理 |
| Data Center | 本地数据、正式环境与测试环境管理 |

## 插件生态

Nexus Ones 正在构建插件系统。

未来将支持：

- 第三方插件
- 个人插件
- 社区插件
- AI 插件
- 插件开发说明书
- 插件安装与卸载规范

插件系统当前处于骨架阶段，插件开发文档将在后续版本补充。
